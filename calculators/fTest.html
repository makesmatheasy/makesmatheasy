F-Test

A Statistical F Test uses an F Statistic to compare two variances, s1 and s2, by dividing them.

The result is always a positive number (because variances are always positive).

The equation for comparing two variances with the f-test is: F = s1^2 / s2^2 ( if s1^2> s2^2).

Any statistical test that uses F-distribution can be called an F-test. It is used when the sample size is small i.e. n < 30.


Formula for f-value:


(Larger Sample Variance)/(Smaller Sample Variance)
