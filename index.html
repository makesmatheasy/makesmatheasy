    Properties of Matrices

    1. If the rows and columns in a matrix A are interchanged, the new matrix is called the transpose of the original matrix The transposed matrix is denoted by AT

    2. A square matrix A is called orthogonal if A AT = I where I is identity matrix

    3. If the matrix product AB is defined, then (AB)T = BTAT

    4. AA-1 = I where A-1 is inverse of matrix A

    5. (AB)-1 = B-1A-1

    6. A matrix is said to be Involutary if A2 = I

    7. A matrix is said to be Idempotent if A2 = A

    8. A matrix is said to be Nilpotent if Ap = 0 , where p is any positive integer

    9. A matrix is said to be symmetric if A = AT and skew-symmetric if A = -AT

    10. A matrix is said to be Hermitian if A = A* and skew-Hermitian if A = -A* where A* is transpose conjugate of matrix

    Properties of Determinants

    1. det(XY) = det(X)det(Y) where det is determinant

    2. det(X) = det(XT)

    3. det(X-1) = -det(X)

    4. The determinant of a matrix will be 0 if any one column or row is 0

    5. In a triangular matrix , the product of diagnol elements is determinant

    6. A matrix A can be written as det(A- Î»I) = 0 , the Î» are eigen values of the matrix
